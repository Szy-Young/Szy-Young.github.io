Ziyang Song

I am currently a 2nd-year PhD student at The Hong Kong Polytechnic University, supervised by Prof. Bo Yang.

Previously, I got my M.Eng and B.Eng degrees (Honors Youth Program) from Xi'an Jiaotong University, supervised by Prof. Zejian Yuan. During my M.Eng study, I interned at SenseTime with Dongliang Wang, and Tencent Robotics X with Wanchao Chi.

My general research interests lie in computer vision and machine learning. Currently I am focusing on unsupervised / weakly-supervised methods for 3D object segmentation.

Publications
OGC: Unsupervised 3D Object Segmentation from Rigid Dynamics of Point Clouds
Z. Song, B. Yang
Advances in Neural Information Processing Systems (NeurIPS), 2022
arXiv / Video / Code

We propose the first unsupervised 3D object segmentation method, learning from dynamic motion patterns in point cloud sequences.

ActFormer: A GAN Transformer Framework towards General Action-Conditioned 3D Human Motion Generation
Z. Song, D. Wang, N. Jiang, Z. Fang, C. Ding, W. Gan, W. Wu
arXiv

A GAN Transformer framework for general action-conditioned 3D human motion generation, including single-person actions and multi-person interactive actions.

Learning End-to-End Action Interaction by Paired-Embedding Data Augmentation
Z. Song, Z. Yuan, C. Zhang, W. Chi, Y. Ling, S. Zhang
Asian Conference on Computer Vision (ACCV), 2020
CVF Open Access / arXiv

We specify a new task to learn end-to-end action interaction from unlabeled interactive action pairs, and a Paired-Embedding (PE) data augmentation method for efficient learning with small data.

Attention-Oriented Action Recognition for Real-Time Human-Robot Interaction
Z. Song, Z. Yin, Z. Yuan, C. Zhang, W. Chi, Y. Ling, S. Zhang
International Conference on Pattern Recognition (ICPR), 2020
IEEE Xplore / arXiv / code

A framework for real-time human action recognition from RGB-D videos in human-robot interaction (HRI) scenarios.
